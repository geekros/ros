## Install

```shell
pip install geekros
# or
pip3 install geekros
# or
python3 -m pip install geekros
```

## Build

> Build and publish ROS Node Package.

```shell
git clone git@github.com:geekros/ros.git
```

```shell
cd ros
```

```shell
sudo ./publish.sh
```
